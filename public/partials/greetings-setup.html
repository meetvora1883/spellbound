📚 Available Variables
👤 User Variables
{mention} – Mentions the user (@username)
{user} – User's username
{user(id)} – User's Discord ID
{user(proper)} – User's tag (Username#1234)
{user(avatar)} – User's avatar URL
{user(created)} – When user created their account
{user(joined)} – When user joined the server
🏠 Server Variables
{server} – Server name
{server(id)} – Server ID
{server(members)} – Total member count
{ord:{server(members)}} – Member count with ordinal (1st, 2nd, 3rd)
{time} – Current time
{date} – Current date
{event} – Event type (welcome/farewell/ban/kick)
📊 Invite Tracking Variables
{inviter} – Username of who invited the user
{inviter(mention)} – Mention of who invited the user
{invites} – Total invites by the inviter
{invites(regular)} – Regular invites (still in server)
{invites(leave)} – Invites who left
{invites(fake)} – Fake invites (bots)
🎲 Advanced Variables
{random:option1~option2~option3} – Randomly selects one option
{math:2+2} – Performs mathematical calculations

Examples:
Welcome {mention}! You are member #{ord:{server(members)}}
{random:Hey~Hello~Greetings} {user}, enjoy your stay!
You were invited by {inviter(mention)} who has {invites} invites!
